### Linux配置说明

1. vim安装markdown插件:
   
   采用vundle管理插件，修改配置文件
   ```vim
   Plugin 'godlygeek/tabular'
   Plugin 'plasticboy/vim-markdown'
   ```

   之后执行
   ```vim
   :PluginInstall
   ```

   预览插件vim-instant-markdown
   需要先安装nodejs和npm,然后安装vim插件

   ```vim
   Plugin 'suan/vim-instant-markdown'
   ```
   之后在vim里面执行:
   ```vim
   :PluginInstall
   ```
